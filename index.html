LONG-DIVISION TRACE

#brvv≥3qr′
Each step slides one bit of N into the 3-bit window v=(r≪1)|b. Since r<3, v≤5, so the quotient digit is a single bit: v≥3 → q=1, r′=v−3, else q=0, r′=v. The comparison and subtraction are baked into the NOT/AND/OR gates — no arithmetic unit anywhere.

🤝 async: one stage, no clock — every signal rides two rails (x.t, x.f) so the data carries its own validity; a C-element completion detector fires DONE exactly when the gates finish, the latch captures r′, and a NULL spacer wave resets the rails for the next bit. Unbounded bits, finite hardware, zero clock. ⏱ clocked: one copy of the stage reused n times via the remainder register — minimal hardware, n cycles. ⚡ no clock: n chained copies of the same stage, no register — all inputs applied at once, the answer settles after one propagation wave down the chain.
